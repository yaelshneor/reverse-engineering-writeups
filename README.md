# reverse-engineering-writeups

This repository contains my reverse engineering analysis of the Wonderland CTF-style challenge.

The challenge is structured into multiple stages, each increasing in complexity and requiring binary analysis, input transformation understanding, and reverse engineering techniques.

---

## 🧩 Structure

Each stage folder contains:
- Full reverse engineering writeup
- Assembly analysis
- Explanation of logic
- Python solving script (when applicable)
